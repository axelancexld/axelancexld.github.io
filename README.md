# MellyCraft SMP Website

A professional, multi-language website for the MellyCraft Minecraft modpack.

## Features

- **Multi-language Support**: Turkish, English, and German
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop
- **Modern UI**: Clean, professional design with Hello Kitty theme
- **Interactive Elements**: Smooth animations, hover effects, and parallax scrolling
- **CurseForge Compatible**: Direct download link integration
- **Easter Egg**: Konami Code for Ultra Kawaii Mode

## Technical Details

- **Minecraft Version**: 26.1
- **Pack Size**: 233KB
- **Mods**: 100+
- **Launcher**: CurseForge

## File Structure

```
mellycraft-website/
├── index.html          # Main HTML file
├── style.css           # Styling and animations
├── script.js           # Interactive functionality
├── translations.js     # Multi-language support
├── mellycraft.png      # Logo
└── README.md          # Documentation
```

## Usage

1. Open `index.html` in any modern web browser
2. Use the language switcher (TR/EN/DE) in the navigation bar
3. Navigate through sections using the menu or scroll
4. Click "Download" to get the modpack

## Language Support

The website automatically saves your language preference in localStorage. Supported languages:

- 🇹🇷 Turkish (Türkçe)
- 🇬🇧 English
- 🇩🇪 German (Deutsch)

## Customization

To update content:

1. **Translations**: Edit `translations.js`
2. **Styling**: Modify `style.css`
3. **Functionality**: Update `script.js`
4. **Download Link**: Change the href in `index.html` (line with download button)
